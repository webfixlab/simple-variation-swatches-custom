A truly lightweight EASY to use and super FAST WooCommerce variation swatches solution to replace default variation dropdown with button, color, image & radio button fields.

== Description ==

Say goodbye to boring old dropdown variations in WooCommerce and hello to lightning-fast, user-friendly swatches. This straightforward plugin is packed with everything you need to replace boring old dropdowns with buttons, colors, images, and radio buttons.

The best part? No complicated features or slow page speeds here! Get ready for a streamlined and seamless shopping experience.

[Demo](http://demo.webfixlab.com/product/simple-variation-swatches/) | [Docs](https://docs.webfixlab.com/kb/simple-variation-swatches-for-woocommerce/) | [Bug Report](https://wordpress.org/support/plugin/simple-variation-swatches/) | [Customize Request](https://webfixlab.com/contact/)

**FEATURES**

* WooCommerce High-Performance Order Storage (HPOS) enabled
* 5 types swatches: color, image, button, radio button and dropdown
* Option to hide attribute labels
* Option to set default swatch or variation type
* Set color and image swatches size
* 3 styles for color and image swatches: square, circle and round corner
* Set font size for button and radio button
* Truly lightweight and extremely fast, almost no impact on page speed

== Installation ==

1. Upload the plugin folder "simple-variation-swatches" to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Then go to ** Products > Attributes** and select the swatches type

== Frequently Asked Questions ==

= How can I set the swatches? =

Go to ** Products > Attributes** and select the swatches type

= Where is the settings? =

In the left menu bar of the dashboard **Variation Swatch**

= Will it work on any theme or page builder? =

Yes, it will work fine on any standard theme or page builder like Gutenberg, Elementor, DIVI, WPBakery Composer etc unless they manipulate the variations options too.
